## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm i
```

## Running the app

```bash
# watch mode
$ npm run start:dev

# migration mode
$ npm run start:mig
```

## Env
```bash
DATABASE_URL="postgresql://{dbUserLogin}:{dbUserPassword}@{dbIp}:{dbPort}/{dbName}?schema=public"
SALT=0
ACCESS_SECRET=0
REFRESH_SECRET=0
```
